Accuracy and Precision

In physics, we often repeat a measurement several times. Two important ideas help us judge how good these measurements are: accuracy and precision.

Accuracy describes how close a measured value is to the true (accepted) value of the quantity being measured. For example, if the true length of a block is 20.6 cm, a reading of 20.5 cm is more accurate than a reading of 20.4 cm, because it is closer to the true value.

Precision describes how close a set of repeated measurements are to each other, regardless of whether they are close to the true value. A precise set of measurements is tightly grouped together, even if that group is not near the true value.

A good way to picture this is a game of darts. If several darts land close together, the player has thrown precisely. If the darts land close to the bull's eye (the centre), the player has thrown accurately. It is possible to be precise without being accurate, accurate without being precise, both, or neither — as you'll see below.

Remember: accuracy is mostly affected by systematic errors, while precision is mostly affected by random errors and the resolution of your instrument. This distinction matters for your IB Physics uncertainty calculations later in the course.
