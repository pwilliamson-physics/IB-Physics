What the words mean

Random error
Scatters readings either side of the true value (e.g. reaction time, judging a fast event). Reduced by repeating and averaging. Shows as spread.
Systematic error
Shifts every reading the same way (e.g. a zero error, a miscalibrated meter, consistent parallax). Repeating does not help. Shows as offset from the bullseye.
Precision
How tightly repeated readings agree — a small spread. Says nothing about being correct.
Accuracy
How close a reading is to the true value — a small offset.
Resolution
The smallest change an instrument can show (its finest division or last digit). It sets the floor on uncertainty for a single reading.

Uncertainty in a log or ln value OCR method

When you tabulate ln y or log y to plot, each point needs an uncertainty. Because a log scale is not symmetric, OCR averages the upper and lower differences: Δ = ½ ( |f(x+Δx) − f(x)| + |f(x) − f(x−Δx)| ).
