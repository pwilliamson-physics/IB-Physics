Assuming the resistors are identical, group 1 (all three in series) would have the highest resistance, and group 3 (all three in parallel) would have the lowest resistance. Adding a resistor in series always increases the total resistance; adding a resistor in parallel always decreases it (it opens up an extra path for charge to flow).

Modelling resistors: the rope-bridge analogy

Imagine two rope bridges crossing a gorge, one after the other, each letting 20 people across per minute. Because every person must cross both bridges in turn, the two bridges act like resistors in series: putting a second obstacle in the only available path can only make the journey harder overall, so the total "resistance" to the flow of people increases.

Now imagine a second bridge built alongside the first, so people can choose either bridge. There are now two available routes, so more people can cross per minute overall than with a single bridge. This is the parallel case: adding an extra path in parallel makes it easier for charge (or people) to get through, so the total resistance decreases.

Modelling resistors: the water-pump analogy

A pump drives water around a closed pipe loop. A narrow section of pipe restricts the flow, acting like a resistor. If a second narrow section is added in series (one after the other in the same pipe), the water has to squeeze through both restrictions in turn, so the flow rate for a given pump effort drops — the combined resistance to flow is higher than either restriction alone.

If instead a second narrow section is added in parallel with the first (a second pipe alongside, so water can take either route), more water can flow overall for the same pump effort — the combined resistance to flow is lower than either restriction alone.

Part B — Workbook: series and parallel resistors

How do we calculate total resistance in series circuits?

The relationship of the potential differences is known: VT = V1 + V2. A model of an electron travelling around the circuit can be useful here as we can see that the electron must go through both resistors. Since I is the same everywhere in a series circuit, this can be written IRT = IR1 + IR2. Cancelling for current gives:

RT = R1 + R2
R₁ = 2 Ω, R₂ = 4 Ω (in series). What is Rtotal?
Rtotal = Ω
RT = R₁ + R₂ = 2 + 4 = … Ω
R₁ = 6 Ω, R₂ = 34 Ω (in series). What is Rtotal?
Rtotal = Ω
RT = R₁ + R₂ = 6 + 34 = … Ω

How do we calculate total resistance in parallel circuits?

The relationship of the currents is known: IT = I1 + I2. Since the potential difference is the same across each branch in a parallel circuit, this can be written V/RT = V/R₁ + V/R₂. Cancelling for potential difference gives:

1/RT = 1/R₁ + 1/R₂

The notation can be annoying….

For N resistors in parallel, the general rule is:

1/RT = 1/R₁ + 1/R₂ + 1/R₃ + … + 1/RN

Note that RT is always smaller than the resistance of the smallest resistor in the parallel combination.

It is often more convenient to use conductance rather than resistance when dealing with resistors in parallel. The equivalent conductance for N resistors in parallel is:

GT = G₁ + G₂ + G₃ + … + GN,   where G₁ = 1/R₁, G₂ = 1/R₂, … , GN = 1/RN
