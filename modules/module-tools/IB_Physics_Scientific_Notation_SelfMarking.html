Scientific Notation (Standard Form)

Scientific notation (standard form) is used throughout IB Physics to write very large and very small numbers clearly, and to keep track of significant figures correctly. This resource lets you practise the skill and check your own answers as you go.

Please choose a level to begin. If you are not so sure about scientific notation, start at the beginning and complete Task 1. If you are already confident, you can skip straight to Task 2.
